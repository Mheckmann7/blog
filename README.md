<h1 align="center"> Hi Tech </h1>

This is a full-stack blog application giving you the ability to signup, login, post, edit, delete, comment, and favorite.

Deployed website: [Hi Tech](link)

## What did we use?

- Python
- Django
- PostreSQL
- Materialize
